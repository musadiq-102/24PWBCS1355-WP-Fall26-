Task 5 – Creating and Deleting test.txt

Step 1: Create test.txt

Created a file named test.txt with the following content:

    This is a sample test file for Git tracking demonstration.

Commit Message:

    git add test.txt
    git commit -m "Added test.txt file"
    git push origin main

Step 2: Delete test.txt

Deleted the file test.txt.

Commit Message:

    git rm test.txt
    git commit -m "Removed test.txt file"
    git push origin main

Task 9 – Adding a Screenshot

Step 1: Create a folder named screenshots in the Lab 01 directory:

    mkdir screenshots

Step 2: Take a screenshot of:

  • Your Git version in terminal (git --version)
  • OR your Visual Studio Code setup

Step 3: Save the screenshot inside the screenshots folder.

Step 4: Commit and push the screenshot:

    git add screenshots
    git commit -m "Added screenshot in screenshots folder"
    git push origin main

Task 10 – Cloning the Repository to a New Location

Step 1: Open a new folder on your computer (different from your original Lab 01 folder).

Step 2: Clone your GitHub repository into the new folder using the command:

    git clone https://github.com/your-username/your-repository-name.git

Step 3: Verify that the repository has been cloned by checking the folder structure.

Step 4: Take a screenshot of the cloned folder structure.

Step 5: Move the screenshot into the screenshots folder of your original Lab 01 repository.

Step 6: Add, commit, and push the screenshot to GitHub:

    git add screenshots
    git commit -m "Added cloned repository screenshot"
    git push origin main
